Introduction to HTML by Rosaleen Walsh

Doctype

All documents start with a <!Doctype> declaration. It is information to the browser about what document type to expect. To show that it is a HTML document we write <!Doctype html>.

html

We use a < html></html> tag to surround all of our HTML coding. All coding apart from the doctype declaration goes between these tags.  

head

The <head></head> tag is placed between the <html> opening tag and the <body> opening tag. It contains metadata, i.e. data about data. It can contain items such as styles, title, description.

body

The <body></body> tag contains all of the contents of the webpage. It contains headings, paragraphs, images, links etc. It is what can be seen when a website is opened up.   

title

The <title></title> tag is contained within the head of the coding. The text that is written within the title is what comes up in a browser's title bar.  

Example:

 <title>Introduction to HTML by Rosaleen Walsh</title>

p

The <p></p> tag defines a block of text on a webpage. Each paragraph starts on a new line. 

<p>this is a paragraph</p> 

Here is an example of a paragraph:

Lorem ipsum dolor sit amet, consectetur adipisicing elit. Minima nobis repellendus soluta quae, quam placeat, sequi quibusdam aspernatur. Atque, cumque.

span

 <p>This is an <span style="font-size:200%"> example of span </span>within a paragraph.</p> 

This is an example of span within a paragraph. 

It is an inline tag that can be used to style a specific portion of text within a paragraph.  

div

A <div></div> is used to separate a website into sections. It can then be styled inline or by using classes and ids.

<div style="font-size=150%>This is my first div</div>

<div style="font-size=70%>This is my second div</div>

images and Alt

You can insert an image into your website using an <img> tag. there is no closing tag. Within the tag we include a source code "src", which is a pathway to the image in our images folder.

Additionally, within the image tag, we use an "alt" attribute , which provides alternate text if the image doesn't appear. It can also be used by screen readers to enable visually impaired readers to use the page.

Here is an example of coding for an image:

<img src="candles2_small.png" alt="decorative candle">

quote and blockquote

 The quote tag, <q></q>, is used to insert short quotes into a section of text. The text will be surrounded by quotation marks when run. It is written as follows:

 

<p>As Humphrey Bogart once said,  <q>Here's looking at you, kid.</q> </p>

It appears on the page as follows:

 

As Humphrey Bogart once said, Here's looking at you, kid.

 

The <blockquote></blockquote> tag is used for longer quotes and there is generally a citation included within the tag. It appears indented on the webpage. Here is how it is written:

 

<p>Chief Brody's famous quote from the Jaws movie:         

<blockquote cite="https://www.rottentomatoes.com/m/jaws/quotes/">

You're gonna need a bigger boat!</blockquote> </p>

 

This is how it looks on the webpage:

 

Chief Brody's famous quote from the Jaws movie:

You're gonna need a bigger boat!

 

 

abbreviations

The <abbr></abbr> tag defines an abbreviation within a text. It is accompanied by a title attribute where the full version is written. The code is written as follows:  

<p>

<abbr title="Radio Teilifis Eireann">R.T.E.</abbr> is Ireland's national broadcasting service.

</p> 

It appears as follows: 

R.T.E. is Ireland's national broadcasting service.

 The title attribute allows us to see the full version when we hover over the abbreviation.

meta

The <meta> tag is used to describe the metadata in a document. Metadata is defined as data about data. The tag is usually accompanied by "name" and "content" attributes. Here are some examples:

 <meta name="author" content="Rosaleen Walsh">

This meta gives the author's name.

<meta name="description" content="Level 5 Web Authoring Assignment>

This meta gives a brief description of the contents of the website.

<meta name="keywords" content="HTML, Assignment, tags, definitions">

This meta provides keywords for the webpage.

<meta http-equiv="refresh" content="30">

This meta causes the webpage to refresh every 30 seconds.

 Metadata is not seen on the webpage but provides information for browsers and search engines.

Special symbols and character entities:

There are some symbols that can't be found on a keyboard, for example the trademark symbol, some maths symbols, or the copyright symbol. Other symbols, such as the Euro character, can be found on the keyboard but older browsers may not be able to read them. In these cases, we can use the entity name or hexadecimal number for the symbol. This ensures that the symbol is always displayed correctly. Here are some examples:
